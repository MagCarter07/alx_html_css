# alx_html_css

This project contains HTML and CSS files for learning the basics of web development.

## Project Structure

- html_basic/ — HTML files from the previous project  
- css_basic/ — New folder containing:
  - index.html
  - tweets.html
  - base.css
  - styles.css

## Purpose
The goal of this project is to learn:
- Basic HTML structure
- Adding CSS styling
- Organizing files and directories
- Following ALX front-end project requirements

## Requirements
- No external libraries (no Bootstrap, React, Vue, etc.)
- W3C-compliant HTML
- File names must end with a newline
